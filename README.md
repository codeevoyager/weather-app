# Weather app

Created Weather app by HTML, CSS and Javascript using OpenWeatherMap API. 

when you search the city name it will give details of the city's weather data

- city name
- weather-main
- weather-description
- weather-icon
- main-temperature
- min-temperture
- max-temperature
